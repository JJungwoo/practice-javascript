# CSS 처음 사용할 때 꼭 알아두어야 할 것들

## CSS 적용하는 3가지 방법

**1. inline 방식**
  - htnml 태그에 직접 선언

```html
<!-- h1에 css 적용 -->
<h1 style="color:blue"> TEST </h1>
```

**2. internal 방식**
  - html 파일의 body 태그가 아닌 head 태그 사이에 선언

```html
<head>
    <style>
        /* h1 태그에 모두 스타일 적용 */
        h1 {
            style="color:blue"
        }
    </style>
</head>
<body>
    <h1> TEST </h1>
</body>
```

**3. external 방식**
  - 타겟 html 파일 외부에 파일로 선언

```html
<!-- todolist.css 파일을 읽어올 때 예시 -->
<link href="todolist.css" rel="stylesheet" type="text/css">
```

## 주요 선택자

### 1) 전체 선택자
HTML 페이지에 있는 모든 요소를 대상으로 스타일을 적용할 때 사용한다.

- 문법

```css
* { 스타일 }
```

- 예시

```css 
* {
    margin:0;
    padding:0
}
```

### 2) 태그 선택자
HTML 페이지의 특정 태그를 사용한 모든 요소에 스타일 적용할 때 사용한다.

- 문법

```css
태그 { 스타일 } 
```

- 예시

```css
h1 {
    color: green
}
```

### 클래스(class) 선택자
HTML 페이지에 여러번 반복할 스타일이 특정 부분으로 선택하여 적용할 때 사용한다.
클래스 선택자는 식별자 앞에 '.'을 사용하여 선언할 수 있다.

- 문법

```css
.식별자 { 스타일 }
```

- 예시

```css
.tc1 { 
    color: yellowgreen;
}
```

### id 선택자
HTML 페이지에서 한번만 스타일을 적용할 부분을 선택하여 사용한다.
id 선택자는 식별자 앞에 '#'을 사용하여 선언할 수 있다.

- 문법

```css
#식별자 { 스타일 }
```

- 예시

```css
#t1 { 
    color: yellowgreen;
}
```

### 클래스와 id 선택자 차이
클래스 선택자는 여러번 사용될 수 있지만 id 선택자는 단한번만 하나의 태그에만 사용될 수 있다.

### 그룹 선택자
같은 스타일을 사용하는 선택자를 한번에 묶어서 정의하여 사용할 수 있다.
',' 값을 통해 선택자를 여러개 그룹 선택할 수 있다.

- 문법

```css
태그1, 태그2 { 스타일 }
```

- 예시

```css
h1, h2 { 
    color: blue;
}
```

> 태그 선택자 외에도 class, id 선택자 모두 사용 가능하다.

## 선택자 속성

### 연결 선택자

선택자와 선택자를 연결해 적용 대상을 한정하는 선택자
컴비네이션 선택자(combination selector) 또는 '조합 선택자'라고 함

#### 하위 선택자

상위 요소에 포함된 모든 하위 요소에 스타일 적용(자식뿐만 아니라 자손까지 모두 적용)

- 문법 

```css
상위선택자 하위선택자 { 스타일 }
```

- 예시

```css
#board li {
    color: blud;
}
```

#### 자식 선택자

자식 요소에 스타일을 적용하는 선택자(하위 요소중에 특정 자식 요소만 스타일을 적용, 자손까지는 적용 안됌)

- 문법 

```css
상위선택자 > 하위선택자 { 스타일 }
```

- 예시

```css
#board > li {
    color: blud;
}
```

#### 인접 형제 선택자

같은 부모를 가진 형제 요소 중 자신 다음으로 첫 번째 형제(동생) 요소에만 스타일 적용(자신에게는 스타일 적용 안함)

- 문법 

```css
선택자 + 선택자 { 스타일 }
```

- 예시

```css
h1 + li {
    color: blud;
}
```

> h1 다음에 오는 첫 번째 li 태그에만 스타일 적용 다음으로 li이 또와도 적용하지 않는다.

#### 형제 선택자

형제 요소들에 스타일 적용(인접 형제 선택자와는 다르게 모든 형제 요소에게 모두 적용)

- 문법 

```css
선택자 ~ 선택자 { 스타일 }
```

- 예시

```css
h1 ~ li {
    color: blud;
}
```

> h1 다음에 오는 li 태그 요소들 모두 스타일 적용시킨다.

### 속성 선택자

지정한 속성을 가진 요소를 찾아 스타일 적용, 속성 값의 조건에 따라 다양한 스타일을 지정할 수 있어 활용도가 높음

- E[속성] : 지정한 속성이 포함된 요소를 선택
- E[속성 = 값] : 속성의 값이 값에 일치하는 요소를 선택
- E[속성 ~= 값] : 여러 값들 중 특정 값이 포함되어 있는 요소를 선택 (공백으로 분리된 값이 일치 해야함)
- E[속성 |= 값] : 값이 포함되어 있는 요소를 선택(값으로 시작 혹은 정확히 값이 들어갈 때)
- E[속성 ^= 값] : 값으로 시작하는 요소를 선택
- E[속성 $= 값] : 값으로 끝나는 요소를 선택
- E[속성 *= 값] : 값이 속성의 값 일부로 포함되는 요소 선택

## CSS 우선 순위

### CSS 적용 계단식 (cascade) 우선순위

css 에서 계단식 우선 순위는 css 스타일이 중복 되었을 때, css의 가장 마지막에 나오는 스타일이 적용되는 것을 말한다.

```css
h1 { 
    color: red; 
}
h1 { 
    color: blue; 
} 
```

```html
<h1>계단식 우선순위 적용 테스트</h1>
```

위의 h1 태그는 css 마지막에 선언된 blue로 스타일이 적용된다.

### CSS 선택자 적용 우선순위

같은 요소에 선택자가 겹칠 때 우선 순위는 다음과 같다.

1. inline 스타일 
2. id 스타일
3. class 스타일
4. tag 스타일

## CSS 스타일 상속(Inheritance)

부모 요소에 적용된 스타일은 자식 요소에 상속되어 적용된다.
상속 속성 설정이 가능하다. 하지만 그 내용까지는...

> 일부 속성은 상속되지 않는다. 
ex) width를 50%로 설정한다고 모든 자식 요소가 50%로 변하지 않는다.
